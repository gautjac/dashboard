X Bookmarks Settings

Daily Dashboard Extension

Generate an API key in your Dashboard Settings under Bookmarks > Browser Extension

Only change this for local development

How it works

  1. When you bookmark a tweet on X, the extension captures it
  2. Bookmarks are sent to your Daily Dashboard
  3. They appear in your Bookmarks widget instantly
